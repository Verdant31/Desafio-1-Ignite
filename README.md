## ⚙️ Instalação
```
# Abra um terminal e copie este repositório com o comando
$ git clone https://github.com/Verdant31/Desafio-1-Ignite.git
```

```
# Acesse a pasta da aplicação
$ cd Desafio-1-Ignite

# Instale as dependências
$ yarn

# Inicie a aplicação
$ yarn start

```
